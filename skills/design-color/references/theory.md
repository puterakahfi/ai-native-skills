# Color Theory Foundations

> Hue, value, saturation — the three axes of all color decisions.

## The Three Axes

```
HUE         — position on color wheel (red, orange, yellow, green, blue, violet)
VALUE       — lightness/darkness (L in HSL — 0=black, 100=white)
SATURATION  — intensity/purity (S in HSL — 0=gray, 100=full color)

Every color decision = a choice on all three axes simultaneously.
```

## Color Relationships (Harmony)

```
MONOCHROMATIC   — same hue, varying value + saturation
  Use: zen, minimalist, editorial — restrained, cohesive
  Example: #0c0b09 → #4a4840 → #c4c0b4 (warm dark to light)

ANALOGOUS       — 2–3 adjacent hues on wheel (30–60° apart)
  Use: atmospheric, nature — harmonious, low contrast
  Example: teal + blue + indigo

COMPLEMENTARY   — opposite hues (180° apart)
  Use: high contrast accent — punch, energy, tension
  Example: deep blue bg + amber accent

SPLIT-COMPLEMENTARY — one hue + two adjacent to its complement
  Use: more sophisticated than complementary, less tension
  Example: deep blue + warm yellow + warm orange

TRIADIC         — three evenly spaced hues (120° apart)
  Use: playful, balanced — vibrant but not harsh
  Example: red + yellow + blue
```

## Value Contrast

```
Dark palette (L < 20%):    bg = L10, surface = L15, ink = L85-95
Light palette (L > 85%):   bg = L97, surface = L100, ink = L5-15
Mid palette (L 40–60%):    avoid — neither crisp nor dramatic

WCAG contrast (handled by readability skill):
  Primary text on bg: ≥ 4.5:1
  Secondary text:     ≥ 3.0:1
  Large text (24px+): ≥ 3.0:1
```

## Saturation by Genre

```
ZEN/MINIMALIST:    S 0–15% — near-neutral, warm gray. Accent S 30–40% (sage)
EDITORIAL:         S 0–20% — muted, paper-like. Accent S 40–60%
MODERN MINIMAL:    S 5–25% — clean neutral. Accent S 70–90% (indigo/blue)
ATMOSPHERIC:       S 60–90% — rich darks. Accent S 80–100% (glowing)
PLAYFUL:           S 50–80% — warm/bright. Multiple accents ok (max 3)
```
