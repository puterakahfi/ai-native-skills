# Genre: Playful

> Warm, encouraging, human — joy in every interaction.

## Signal Words
```
fun, consumer, casual, friendly, onboarding, family,
community, kids, social, lifestyle, wellness
```

## Voice & Copy
```
Voice:     Warm, encouraging, human
Copy tone: Conversational, first/second person, light humor acceptable
           "You've got this." — direct, friendly, not corporate
```

## Typography
```
Pair:    Rounded sans or humanist sans
H1:      medium-large, playful weight variation
Body:    casual leading (1.6–1.75), friendly measure
```

## Color
```
Warm or saturated palette — not monochromatic
Multiple accent colors acceptable (max 3, semantic roles)
Light mode primary — dark mode needs explicit intent
```

## Layout
```
Softer edges (border-radius generous)
Illustrations > photography
Confetti, texture, pattern: acceptable in small doses
```

## Macrostructures
```
Hum, Carnival, Garden, Riso, Sport (light), Bento (playful variant)
```

## Nav & Footer
```
Nav:    floating pill (N5), floating chip (N2), scroll-morph (N10)
Footer: warm (Ft7), minimal (Ft1)
```

## Slop Gates
```
❌ Corporate language ("leverage", "enterprise-grade")
❌ All-dark color palette
❌ Photo of person in a suit
✅ Illustrations, icons, emoji encouraged (not required)
```
